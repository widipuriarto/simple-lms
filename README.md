# Simple LMS - Django Docker Setup

## 📌 Deskripsi
Project ini adalah setup awal Simple LMS menggunakan:
- Django
- PostgreSQL
- Docker & Docker Compose

---

## 🚀 Cara Menjalankan

1. Clone repository
2. Copy environment:
   cp .env.example .env

3. Jalankan:
   docker-compose up -d

4. Buka browser:
   http://localhost:8000

---

## ⚙️ Environment Variables

- DB_NAME → nama database
- DB_USER → user database
- DB_PASSWORD → password database
- DB_HOST → host database (db)
- DB_PORT → port database (5432)
